Regular markdown still works exactly as before. You can write **bold**, _italic_,
`inline code`, and all the usual stuff.

## Using Callouts

<Callout type="info">
  This is an info callout. Great for side notes or context.
</Callout>

<Callout type="tip" title="Pro tip">
  Use `tip` callouts for actionable advice.
</Callout>

<Callout type="warning">
  Warning callouts draw attention to things to watch out for.
</Callout>

<Callout type="danger" title="Do not do this">
  Danger callouts for genuinely bad ideas.
</Callout>

## Pull Quotes

## Raw HTML works too

You can drop in raw HTML anywhere:

<div style="display: grid; grid-template-columns: 1fr 1fr; gap: 1rem; margin: 1.5rem 0;">
  <div style="background: rgba(110,86,207,0.1); border: 1px solid rgba(110,86,207,0.2); border-radius: 8px; padding: 1rem;">
    Column one
  </div>
  <div style="background: rgba(0,229,160,0.08); border: 1px solid rgba(0,229,160,0.15); border-radius: 8px; padding: 1rem;">
    Column two
  </div>
</div>
